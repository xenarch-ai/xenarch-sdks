# xenarch-sdks

Official SDKs for the Xenarch payment network.

## JavaScript / TypeScript

```bash
npm install xenarch
```

```typescript
import { Xenarch } from 'xenarch';
const client = new Xenarch({ apiKey: '...' });
```

## Python

```bash
pip install xenarch
```

```python
from xenarch import Xenarch
client = Xenarch(api_key="...")
```

## CLI

```bash
npx xenarch-cli
```
